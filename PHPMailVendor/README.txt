# PHPMailer Samples

### Requerimientos

* [Composer](https://getcomposer.org/)

### Instalar dependencias

Para instalar las dependencias ejecutar en línea de comandos:

```zsh
composer install
```

### Estructura de Archivos

* `sample-gmail.php` envío de correo utilizando servidor SMTP de Gmail.
* `sample-smtp.php` envío de correo utilizando servidor SMTP.
